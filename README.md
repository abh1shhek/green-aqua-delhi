# Green Aqua Delhi 🌱

A marketing site for Green Aqua Delhi — a Delhi-based aquascaping studio specializing in planted aquariums, aquatic plants, and premium livestock.

Live site: https://www.greenaquadelhi.in

## Features

- Full-bleed hero with mouse-tracking parallax and a zoom-out entrance animation
- Scroll-triggered reveal animations, staggered per section
- Frosted-glass navigation header and pill components
- Magnetic hover effect on buttons and nav links
- Tabbed "What We Offer" showcase with cross-fading backgrounds
- Responsive mobile navigation (rounded panel, outlined links)
- No build step, no dependencies — plain HTML/CSS/JS

## Tech

- Vanilla HTML, CSS, and JavaScript (no framework, no bundler)
- Google Fonts: Bricolage Grotesque (display) + Inter (body)
- IntersectionObserver for scroll reveals
- CSS custom properties for theming

## Project structure

```text
├── index.html          # markup only
├── css/
│   └── styles.css      # all styles
├── js/
│   └── script.js       # all scripts
└── assets/             # site imagery
```
## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `main` redeploys automatically.
